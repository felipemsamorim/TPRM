O banco utilizado é o PostGresql, logo faz-se necessário a instalação da versão mais recente no servidor onde ficará o banco.
Como foi utilizado o 'Code First' não precisa criar banco e nem restaurar nenhum bkp ou dump.
Existe um arquivo 'Conexão.config' que armazena a string de conexão com o banco Postgresql e deve ser configurado corretamente antes de executado o projeto
